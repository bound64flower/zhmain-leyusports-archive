# zhmain-leyusports-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态方式存放，不依赖特定后端服务，方便直接通过 GitHub Pages 或其他静态托管服务访问。

## 目录说明

仓库根目录下存放各 HTML 文件及其相关资源（如 CSS、JavaScript、图片等）。每个 HTML 文件通常为一个独立的页面单元，包含完整的前端内容。

```
/
├── index.html          # 入口页面（可选）
├── page-example.html   # 示例页面
├── assets/             # 公共资源目录（可选）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md
```

具体页面文件可能随时间增加或调整，请以仓库实际文件列表为准。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，不包含服务器端逻辑。
- 页面之间的链接、样式、脚本等资源引用均为相对路径，确保在本地或静态托管环境中正常工作。
- 部分页面可能引用外部 CDN 资源（如字体、图标库），请确保网络可达。
- 本仓库不保证所有页面长期可用或内容持续更新，归档性质为主。

## 维护说明

- 欢迎通过 Issue 提交页面内容错误或链接失效的反馈。
- 如需新增或修改页面，请直接提交 Pull Request，建议遵循现有文件结构。
- 仓库维护者会不定期检查并合并合理的贡献。
- 本仓库不涉及用户数据收集或跟踪，纯为静态内容展示。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 HTML 文件即可浏览对应页面。
3. 若需部署到 GitHub Pages，请在仓库设置中启用 Pages 功能，并选择 `main` 分支作为源。
